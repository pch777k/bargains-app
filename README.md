# bargains-app
https://bargainsdemo.herokuapp.com/

Bargains app created with Spring. 

## Tools
#### Backend technologies
- Java
- Spring Boot
- Project Lombok

#### Frontend technologies

- HTML, CSS, JavaScript
- Thymeleaf
- Bootstrap 4

#### Database
- PostgreSQL
